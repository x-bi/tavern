# 阶段 43｜数据备份导出

## 快速读取

- 波次: 世界书、导入导出与设置
- 阶段目标: 提供应用级备份导出能力。
- 前置依赖: 阶段 42。
- 文件范围: backups/export 、设置/备份页。
- 本阶段要做: 导出数据库快照或逻辑导出 JSON。
- 本阶段不要做: 不做异步大任务队列。
- 验收标准: 1. 能导出应用级备份文件 2. 备份文件结构清晰 3. 敏感信息处理方式明确
- 完成后输出: 1. 修改文件列表 2. 新增文件列表 3. 如何测试导出 4. 已完成内容 5. 未完成内容 6. 风险和 TODO
